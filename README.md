# 🪞 FitMirror — 虚拟试穿

上传你的衣服，在3D虚拟人物上实时预览穿搭效果。

**打开即用，无需安装。**

## 功能

- 👕 **上传衣服** — 支持帽子、上衣、裤子、鞋子四个分类
- ✂️ **智能裁剪** — 内置裁剪工具，拖拽框选、旋转、网格辅助线
- 🧍 **3D虚拟人物** — 成人比例、可爱风格，360°拖拽旋转查看
- 💨 **待机动画** — 呼吸起伏、眨眼、摸头发、身体微晃
- 📱 **手机适配** — 移动端和桌面端自动调整布局

## 技术栈

Three.js · Canvas API · ES Modules · GitHub Pages

纯前端，无服务器，所有处理在浏览器完成。

## 快速开始

直接在浏览器打开 `index.html`，或部署到 GitHub Pages：

1. Fork 本仓库
2. Settings → Pages → Source 选 `main` 分支 → Save
3. 几分钟后访问 `https://你的用户名.github.io/fitmirror`

## 使用

1. 点击「+ 添加衣服」选择照片
2. 选择衣服类别（帽子/上衣/裤子/鞋子）
3. 在裁剪框中调整范围，旋转到合适角度
4. 确认后自动出现在虚拟人物身上
5. 拖拽旋转人物，360°查看效果
6. 键盘 1-4 快速切换衣柜分类
